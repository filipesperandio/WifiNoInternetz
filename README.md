# WifiNoInternetz

Simple indicator to your connectivity status. Your wifi may be on, but do you haz internetz?

![show-case](WifiNoInternetz.gif)

Only tested on MacOS Sierra, but may work on other platforms.

# Development

## How to run?

```shell
make start
```

## How to package?

```shell
make installer
```

### Optionally

```shell
make package
make install
make uninstall
```
